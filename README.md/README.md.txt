# Laboratório de Aprendizagem – Análise de Dados com Python

Este repositório reúne os estudos e exercícios práticos desenvolvidos durante minha formação em Análise de Dados com Python, com foco na manipulação, exploração e interpretação de informações para apoiar a tomada de decisão.

## Objetivos

- Desenvolver habilidades em Python para análise de dados.
- Manipular conjuntos de dados utilizando Pandas.
- Explorar técnicas de limpeza e preparação de dados.
- Aplicar conceitos estatísticos básicos.
- Construir análises exploratórias e visualizações.
- Integrar SQL e Python em fluxos analíticos.

## Ferramentas

- Python
- Pandas
- NumPy
- SQLite
- Matplotlib
- Plotly
- Jupyter Notebook

## Competências desenvolvidas

- Análise Exploratória de Dados (EDA)
- Limpeza e tratamento de dados
- Estatística descritiva
- Feature Engineering
- Integração SQL + Python
- Visualização de dados
- Storytelling com dados

## Aprendizados

Ao longo deste laboratório, compreendi que dados isolados têm pouco valor quando não são organizados e contextualizados. Mais do que aprender comandos em Python, desenvolvi uma visão analítica voltada para transformar informações em conhecimento acessível e útil para apoiar decisões.

## Estrutura do repositório

- `notebooks/` → notebooks e exercícios práticos
- `docs/` → documentação do laboratório
- `imagens/` → gráficos, dashboards e capturas de tela
- `certificados/` → certificado da formação
- `datasets/` → informações sobre as bases de dados utilizadas
